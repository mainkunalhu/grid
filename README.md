# Grid

This is simple tool for terminal to show system stats. It show RAM, CPU and Disk usage with progress bar. It look very nice and update in real time.

## How to Build

If you want build it from git clone, you can use these commands. You need Go installed.

First clone it:

```bash
git clone [https://github.com/yourusername/grid.git](https://github.com/yourusername/grid.git)
cd grid

```

Then run one of these command:

- **Run dev mode:** `make dev`
- **Build file:** `make build`
- **Build for Linux:** `make build-linux`
- **Build for Windows:** `make build-window`
- **Build for Mac:** `make build-mac`
